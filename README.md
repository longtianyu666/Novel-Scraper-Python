```
NovelScrape - 轻小说爬取工具 (LiNovelLib)

项目简介 (Introduction)

NovelScrape 是一个高性能的 Python 异步爬虫工具，专为从 [哔哩轻小说 (LiNovelLib)](https://www.linovelib.com/) 网站爬取轻小说内容而设计。

本项目实现了并发下载，还集成了对网站特有 反爬机制（段落乱序） 的逆向解析和还原，确保爬取内容的完整性和正确性。

核心功能

* 异步高效： 使用 aiohttp 实现高并发网络请求，大幅提升爬取速度。
* 反爬处理： 段落乱序还原逻辑，保证章节内容可读。
* 内容完整： 自动下载章节内容及内嵌插图。
* 格式友好： 将爬取的内容自动打包为标准的 EPUB 电子书格式，便于阅读器导入。
* 增量更新： 支持对已下载的小说进行目录和章节的增量更新。
* 图形界面： 提供基于 tkinter 的简单易用 GUI，无需命令行操作。
```
### 1. 环境准备

确保您的系统已安装 Python 3.8 或更高版本

### 2. 安装依赖

本项目依赖一些第三方库。打开命令行或终端，切换到项目根目录，通过 `requirements.txt` 文件安装所需库：
```
pip install -r requirements.txt
```
### 3. 运行程序

直接运行 `GUI.py` 启动图形用户界面：

Bash

```
python GUI.py
```

在 GUI 界面中，输入小说 ID 或目录链接即可开始爬取。

## 文件结构说明 (Project Structure)

**文件名**

**作用**

`GUI.py`

项目的 **图形用户界面** 主入口，负责用户交互和线程管理。

`Scrape_linovelib.py`

爬虫的 **核心调度逻辑**，包含目录解析、章节任务分配和增量更新。

`Scrape.py`

**通用网络请求模块**，封装了 `aiohttp` 的并发控制（信号量）和重试机制。

`unscramble_paragraph.py`

**反反爬模块**，实现了网站特定的段落乱序还原算法。

`Save_data.py`

**数据持久化模块**，负责将爬取数据保存为 **JSON** 和 **EPUB** 格式。

`download_image.py`

图片下载和标准化处理。

`requirements.txt`

项目所需 Python 依赖库列表。

`LICENSE`

项目开源许可证。
